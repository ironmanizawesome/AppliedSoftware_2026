## 수업을 통해 만들어 보고 싶은 소프트웨어
작년 스마트팜창의설계 수업을 들으며 프로젝트 일부로 작물 추천 시스템을 구성한 적이 있었다. 이때 구성했던 시스템은 선택지가 많이 있지도 않아, 결국 유사도를 기반으로 한 시스템이 아니라 단순한 필러팅 시스템에 불과했었다. 

이번에 만약 기회가 된다면, 코사인 유사도 계산을 하고, 이를 바탕으로 비어있는 데이터들도 예측을 하는 시스템까지 구현을 해보면 좋을 것 같다.

다만 조금 걱정이 되는 부분이 있다면, 이는 소프트웨어 자체를 구성하는 문제보다 결국 데이터를 다루는 작업에 신경을 많이 써야 해서 수업의 취지와는 어느정도 동떨어져있다고 생각이 든다.

## 응용소프트웨어개발 수업을 수강신청한 이유
개인적으로, 성격상 정적인 수업보다는 직접 행동하고, 고민하며 생각하는 그런 시간이 주어지는 수업이 나에게 더 적합한 것 같다는 생각을 한다. 단순히 책상에 앉아서, 이론을 듣고 복습하며 외우는 것 보다는 문제가 주어졌을 때 정답을 찾는 활동을 하는것이 더욱 기억에 남고 흥미를 불러일으키는 것 같아 수업을 신청한 이유인 것 같다.
